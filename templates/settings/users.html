Vos collaborateurs

Ici on liste les utilisateurs de vos bailleurs si vous êtes un bailleur, de vos administrations si vous êtes instructeur. De plus, si vous êtes administrateur d'une ou plusieurs entité, pour ces entité, il vous sera possible de : :

  • ajouter un utilisateur
  • modifier un utilisateur
  • supprimer un utilisateur (solf delete à prévoir)
  • nommer un administarteur ou supprimer ce droit d'administration

Si vous n'êtes pas administrateur, alors vous n'avez que le droit de voir la liste de vos collaborateurs
